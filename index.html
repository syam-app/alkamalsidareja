Panduan Penggunaan Aplikasi
1. Menambah Data Alumni

Untuk menambah data alumni:

  1. Buka halaman Database.
  2. Tekan tombol Tambah Data (+).
  3. Isi semua data alumni yang tersedia.
  4. Tekan tombol Simpan.
  • Nama : Nama lengkap alumni
  • Jenis Kelamin : L / P
  • Tahun Lulus : Tahun kelulusan
  • Alamat : Alamat rumah alumni
  • No HP : Nomor WhatsApp alumni
  • Lokasi : Link Google Maps rumah alumni
2. Cara Menambahkan Lokasi Rumah Alumni

Cara 1 (Menggunakan GPS Otomatis)

  1. Tekan tombol Ambil Lokasi.
  2. Izinkan akses lokasi pada browser.
  3. Sistem akan mengambil lokasi GPS Anda.
  4. Link Google Maps akan otomatis muncul di kolom lokasi.

Cara 2 (Manual dari Google Maps)

  1. Buka aplikasi Google Maps.
  2. Cari lokasi rumah alumni.
  3. Tekan dan tahan titik lokasi rumah.
  4. Pilih menu Bagikan.
  5. Salin link Google Maps.
  6. Tempelkan link tersebut di kolom Lokasi.
3. Mengedit / Update Data Alumni

Untuk memperbarui data alumni:

  1. Tekan ikon pada data alumni.
  2. Form edit akan muncul.
  3. Ubah data yang ingin diperbarui.
  4. Jika ingin memperbarui lokasi tekan tombol Ambil Lokasi.
  5. Tekan tombol Update untuk menyimpan perubahan.
4. Menghapus Data Alumni

Untuk menghapus data:

  1. Tekan ikon .
  2. Sistem akan meminta konfirmasi.
  3. Tekan OK jika yakin.
5. Menghubungi Alumni

Untuk mengirim pesan WhatsApp:

  1. Tekan ikon .
  2. WhatsApp akan terbuka otomatis.
  3. Pesan salam akan terisi otomatis.
  • Laki-laki : Assalamualaikum Kang
  • Perempuan : Assalamualaikum Mba
6. Melihat Lokasi Alumni
  1. Tekan ikon .
  2. Google Maps akan terbuka.
  3. Anda bisa melihat lokasi rumah alumni.
7. Pencarian Alumni

Gunakan kolom Search untuk mencari nama alumni secara cepat.

8. Filter Data
  • Filter Desa untuk melihat alumni per desa.
  • Filter Tahun untuk melihat alumni berdasarkan tahun lulus.
9. Dashboard Statistik

Halaman dashboard menampilkan:

  • Total jumlah alumni
  • Jumlah laki-laki dan perempuan
  • Grafik alumni per tahun
  • Grafik alumni per desa

Data statistik akan otomatis diperbarui saat database berubah.
